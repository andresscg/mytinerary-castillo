## Run Locally

Clone the project

```bash
  git clone https://github.com/andresscg/mytinerary-castillo.git
```

Go to the project directory

```bash
  cd mytinerary-castillo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
